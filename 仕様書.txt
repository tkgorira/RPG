--------------------------------------------------------------------------------
1. ゲーム概要
--------------------------------------------------------------------------------
ジャンル       : バレットヘブン / オートバトル ローグライト
プレイ人数     : 1人
目標           : 10分間（600秒）生き残る
操作           : キーボードのみ（マウス：UIクリック可）
開発環境       : Python 3.x + pygame 2.x
実行ファイル   : game.py

--------------------------------------------------------------------------------
2. 操作方法
--------------------------------------------------------------------------------
W / ↑          : 上移動
A / ←          : 左移動
S / ↓          : 下移動
D / →          : 右移動
1 / 2 / 3      : レベルアップ時に強化を選択
M              : BGM・効果音のミュート切替
ESC            : ゲーム終了
ENTER          : ゲームオーバー画面でリスタート
マウスクリック : キャラクター選択 / レベルアップ選択

--------------------------------------------------------------------------------
3. キャラクター
--------------------------------------------------------------------------------
【Knight（騎士）】
  HP    : 200
  速度  : 175
  初期武器 : Axe Lv1
  特徴  : 高い耐久力と防御力。初心者向け。
  外見  : 青い鎧・盾・剣

【Mage（魔法使い）】
  HP    : 90
  速度  : 230
  初期武器 : Wand Lv2（クールダウン短縮済み: 0.38秒）
  特徴  : 低HPだが魔法攻撃が高速。中級者向け。
  外見  : 紫のローブ・三角帽・魔法の杖

【Rogue（ローグ）】
  HP    : 110
  速度  : 290
  初期武器 : Holy Cross Lv1
  特徴  : 最速の移動速度。攻撃的な立ち回りが得意。上級者向け。
  外見  : オレンジのフード・ダガー2本

--------------------------------------------------------------------------------
4. 武器一覧
--------------------------------------------------------------------------------
【Wand（ワンド）】
  - 最も近い敵に向かって魔法弾を発射する
  - Lv が上がるとクールダウン短縮・弾数増加・ダメージ増加・貫通数増加
  - クールダウン : 0.80秒（最短 0.20秒）
  - ダメージ     : 25 + Lv×5
  - 貫通数       : 1 + Lv÷3

【Axe（アックス）】
  - 斜め上方向に扇状に斧を投射、すべての敵を貫通
  - Lv が上がると同時投射数が増える
  - クールダウン : 1.50秒（最短 0.50秒）
  - ダメージ     : 50 + Lv×10

【Holy Cross（ホーリークロス）】
  - 上下左右の4方向に十字弾を発射、貫通
  - クールダウン : 3.00秒
  - ダメージ     : 38 + Lv×8

【Garlic（ガーリック）】
  - プレイヤー周囲にダメージオーラを常時展開
  - 0.5秒ごとに範囲内の敵にダメージ
  - 範囲   : 80 + Lv×20 px
  - ダメージ: 8 + Lv×3

【Lightning（ライトニング）】
  - 最も近い敵にチェーン雷撃を発射
  - Lv+1 体ずつ追加チェーン（最大 350px 範囲内）
  - チェーンするごとにダメージは 60% になる
  - クールダウン : 2.00秒（最短 0.60秒）
  - ダメージ     : 70 + Lv×15（初撃）

【Flame（フレイム）】
  - 足元に炎ゾーンを生成（持続 3秒）
  - 範囲内の敵に 0.3秒ごとにダメージ
  - クールダウン : 4.00秒（最短 1.50秒）
  - 範囲   : 90 + Lv×20 px
  - ダメージ: 18 + Lv×6

--------------------------------------------------------------------------------
5. 強化システム（レベルアップ）
--------------------------------------------------------------------------------
- XP ジェム（水色）を収集するとXPが増える
- XP が規定量に達するとレベルアップ
- 次レベルのXP必要量 = 現在の必要量 × 1.25
- レベルアップ時に3択の強化カードが表示される
- 強化内容 : Wand/Axe/Cross/Garlic/Lightning/Flame の習得またはLv上昇、
             速度+15%、最大HP+30

--------------------------------------------------------------------------------
6. 宝箱システム
--------------------------------------------------------------------------------
- ゲーム開始40秒後から60秒ごとにマップ上にランダム出現
- プレイヤー半径 150〜350px の範囲に配置
- プレイヤーが触れると即時取得
- 報酬内容（ランダム）:
    HP Restore  : HP +60 回復
    Wand Lv+    : ワンドを1段階強化
    Axe         : アックスを1段階強化
    Lightning   : ライトニングを1段階強化
    Flame       : フレイムを1段階強化
    XP Burst    : XP +50
- ボス撃破時にも宝箱を1個ドロップ

--------------------------------------------------------------------------------
7. 敵一覧
--------------------------------------------------------------------------------
【通常敵（ノーマル）】
  HP    : 60 + 経過時間係数×25
  速度  : 90 + 経過時間係数×10
  ダメージ: 15 /秒
  XP    : 5
  出現率: 82%

【高速敵（ファスト）】
  HP    : 28 + 経過時間係数×8
  速度  : 175 + 経過時間係数×22
  ダメージ: 8 /秒
  XP    : 3
  出現率: 18%

【ボス】
  出現タイミング: 2分ごと（120秒・240秒・360秒…）
  HP    : 800 + ボスレベル×300
  速度  : 70 + ボスレベル×10
  ダメージ: 25 /秒
  XP    : 80 + ボスレベル×20
  特殊行動:
    - チャージ攻撃（5秒ごと）: 0.6秒の予告後に猛ダッシュ
    - 出現前10秒から「BOSS INCOMING!」警告表示

  ※ 経過時間係数 = min(経過秒数 / 60, 5)（最大5）

--------------------------------------------------------------------------------
8. 敵スポーン仕様
--------------------------------------------------------------------------------
- プレイヤーから半径 700px の円周上にランダムスポーン
- スポーン間隔: 1.5秒 → 最短 0.3秒（120秒かけて短縮）
- 1回のスポーン数: 1体 → 最大6体（30秒ごとに+1体）

--------------------------------------------------------------------------------
9. ダメージ仕様
--------------------------------------------------------------------------------
- 敵との接触ダメージは DPS 方式（damage × dt / 秒）
- 無敵時間なし → 複数の敵に囲まれると急速にHPが減る
- 被ダメージ効果音は 0.5秒に1回まで（スパム防止）
- HPが0になるとゲームオーバー

--------------------------------------------------------------------------------
10. 視覚・音響効果
--------------------------------------------------------------------------------
【視覚効果】
  - 敵撃破時  : 8個のパーティクルが飛散
  - 被弾時    : 敵が白くフラッシュ
  - ボス警告  : ダッシュ前に赤い波紋を表示
  - ライトニング: チェーンを結ぶ電光ラインを描画
  - 炎ゾーン  : オレンジの半透明円（徐々にフェード）
  - 宝箱      : 金色に光るボブアニメーション
  - 画面シェイク: ライトニング・ボス撃破時

【BGM】
  - Am - F - C - G コード進行のループ（約6.4秒）
  - 手続き的サイン波合成（外部ファイル不要）

【効果音】（手続き的生成）
  shoot / axe / lightning / flame / hit / kill
  levelup / boss / chest / hurt / cross

--------------------------------------------------------------------------------
11. ゲーム進行・勝敗条件
--------------------------------------------------------------------------------
勝利条件 : 600秒（10分）生き残る
敗北条件 : HP が 0 になる

ゲームオーバー画面表示:
  - 生存時間
  - 撃破数
  - ENTER でリスタート（キャラ選択から）/ ESC で終了

--------------------------------------------------------------------------------
12. UI / HUD
--------------------------------------------------------------------------------
- 画面上部中央 : 経過タイマー（MM:SS）
- 画面右上    : キル数
- 画面左上    : 装備中の武器と現在レベル
- 画面下部    : XPバー（水色）＋ レベル・XP表示
- プレイヤー頭上: HPバー（赤）
- 敵頭上      : HPバー（緑）
- 画面右下    : M キーミュート案内
